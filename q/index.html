Supported API calls

/q/ is the base URL for all API calls.
  • hashrate/ - current estimated network hashrate
  • height/ - current height (incl. genesis block)
  • reward/ - current block reward
  • supply/ - circulating supply
  • totalsupply/ - current total supply of coins
  • totalbanked/ - current total banked coins
  • maxsupply/ - max supply for the coin
  • difficulty/ - current difficulty
  • nodecount/ - current node count
  • smartnodecount/ - current smart nodes count
